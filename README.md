I wanted to learn how to create a blog using GitHub.
I had several ideas in mind, and the first approach I tried was creating a simple blog using GitHub Desktop and HTML files.

These are the steps I followed:

Install GitHub Desktop on Windows.

Sign in with your GitHub account.

Create a new repository using GitHub Desktop.

Choose a local folder for the repository.

Publish the repository to GitHub and make sure it is public.

Create an HTML file using Notepad and save it as index.html.

Create additional blog pages like post1.html.

Add a CSS file (style.css) for styling if needed.

Commit the files in GitHub Desktop.

Push the commit to GitHub.

Open the repository online on GitHub.

Go to Settings → Pages.

Select:

Deploy from branch
Branch: main
Folder: / (root)

Save the settings.

Wait a few minutes until GitHub generates the live site URL.

The site will appear at:

https://username.github.io/repository-name/

Whenever you want to add a new post:

Create a new HTML file (for example post2.html)

Add its link in index.html using <a href="">

Commit and push again.

Also make sure:

File extensions are correct (.html, .css)

index.html is in the root folder.
